Если у вас тоже проблемы с выбором, когда вы приходите в Monkey Grinder на ланч...

Телеграм-бот "Random Set in Monkey Grinder" предложит вам случайный набор напиток + перекус из меню кофейни.
Достаточно написать /start для запуска и теперь каждый раз при отправке боту сообщения /bestset вы будете получать случайнейшую прекраснейшую рекомендацию на день. 



Метод запуска:
chmod +x build.sh
sudo /build.sh
